## Trending Workflow 失败排查与修复

### 故障原因
- Run #2 (publish-trending) 失败于 "Publish trending article" 步骤，耗时 37 分钟
- 错误：`urllib.error.HTTPError: HTTP Error 400: Bad Request`
- 位置：publish_to_blogger.py `get_access_token()` → Google OAuth token 端点
- 原因：BLOGGER_REFRESH_TOKEN 失效（Google OAuth refresh token 过期或被撤销）

### 解决方案
- 新增 scripts/refresh_blogger_token.py — 一键重新授权工具
- 流程：读取本地 client_id/secret → 打开浏览器授权 → 粘贴授权码 → 换取新 refresh_token → 更新本地 blogger_tokens.json → 自动推送到 GitHub Secret
- GITHUB_TOKEN 通过环境变量传入（不硬编码，避免触发 GitHub Secret Scanning）

### 用户操作指引
